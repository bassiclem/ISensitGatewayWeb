Welcome to the ISensit Gateway API

Here are a list of things you can do

URIs relative to API Homepage

Resource Method HTTP request Description
index GET /index Homepage
index POST /index/id Add config data
config GET /config Get devices configuration data
config GET /config/id Get gateway configuration data
config POST /config/id Post new gateway configuration data
config DELETE /config/id Delete gateway configuration data
about GET /about About Gateway
get_supported_devices GET /get_supported_devices Get supported devices

ISensit BLE API

Resource Method HTTP request Description
scan GET /ble/scan Scan BLE Devices
connect GET /ble/connect/name Connect to a BLE Device
get_rt_data GET /ble/get_rt_data/name Get BLE Realtime Data
disconnect GET /ble/disconnect/name Disconnect a BLE devices
status GET /ble/status GET BLE Status
